## Como Executar

1. Navegue até a pasta do projeto:
   ```bash
   cd victor-portfolio
   ```

2. Instale as dependências:
   ```bash
   npm install
   ```

3. Execute o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

4. Acesse no navegador:
   ```
   http://localhost:5173
   ```

## Deploy

Para fazer o build de produção:
```bash
npm run build
```

Os arquivos de produção estarão na pasta `dist/`.
